# Informe de Remuneraciones

Pagina estatica publicada en GitHub Pages.

## Actualizar datos

1. En GitHub, abre la carpeta `data`.
2. Reemplaza el archivo `detalle_remuneraciones.xlsx` por el nuevo Excel.
3. Mantén exactamente el mismo nombre: `detalle_remuneraciones.xlsx`.
4. Confirma el cambio en la rama `main`.
5. GitHub Actions regenerara `index.html` automaticamente.

Cuando termine la accion, el mismo link de GitHub Pages mostrara los datos nuevos para todos.

El archivo de carga debe tener las cabeceras en la fila 5 y la hoja `Detalle`.
